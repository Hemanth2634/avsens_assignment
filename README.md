# avsens_assignment

1. select or create a small dataset containing pedestrains and cars
2. Annotate the dataset using a YOLO-compatible annotation format
3. Train a YOLO model for pedestrain and car detection
4. Evaluate the model performance and demoonstrate results on test images


## Submission should include
1. Model performance metrics (precision, recall, mAP)
2. Example detection outputs on test images
3. discussion of failure cases/issues observed (false positives, false negatives, difficult scenarios)
4. A short report explaining you approach and improvements attempted
